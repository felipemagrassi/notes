## Acessing PSQL inside docker

```bash
docker exec -u postgres -it pg psql
```
where `pg` is the name of the container. `-u` indicates a user, `-it` is interactive mode and `psql` is the command to be executed.

## PSQL commands

- `\?` - help
- `\l` - list databases
